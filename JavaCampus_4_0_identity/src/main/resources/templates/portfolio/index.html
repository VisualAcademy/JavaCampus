대표 프로젝트 소개

JavaCampus(DevLec)의 구조를 이해하기 좋은 대표 예제를 간단히 소개합니다.

공지 및 직원 관리 통합 예제

게시판형 CRUD와 관리형 목록 페이지를 함께 확인할 수 있어, Spring MVC와 Thymeleaf 학습 초반에 가장 활용도가 높은 구성입니다.

프로젝트 포인트

  • Spring Boot 4.0.5
  • Thymeleaf Layout Dialect
  • Bootstrap 5 UI
  • JPA + H2 샘플 데이터

활용 아이디어

실습용 사내 포털, 교육용 미니 관리자 사이트, 템플릿 레이아웃 샘플로 확장하기 좋습니다.
