JavaCampus(DevLec) 소개

이 프로젝트는 강의, 스터디, 개인 실습에서 바로 활용할 수 있도록 만든 Spring Boot 학습용 샘플 캠퍼스입니다.

왜 JavaCampus인가요?

복잡한 기능보다 학습 흐름이 잘 보이는 구성을 우선하여, 화면과 백엔드 코드를 함께 익히기 좋게 만들었습니다.

단일 공통 레이아웃, 샘플 시드 데이터, Bootstrap 5 화면, API 메뉴를 통해 기초 구조를 빠르게 파악할 수 있습니다.

2026년 기준 최신 스택 감각에 맞춰 문구와 화면 구성을 정리해 두었기 때문에, 포트폴리오용 연습 프로젝트로도 확장하기 좋습니다.
